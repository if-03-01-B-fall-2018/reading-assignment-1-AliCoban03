1) Word Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?

The Hypertext, hypermedia, HTTP and browsers.

2. How are resources (html pages, images, etc) addressed in the web?

The most common form of URI is the URL (Uniform Resource Locator) (web address). URLs can be typed into your browser's address bar to tell it to load the associated page.

3. What are the general components of a url?

A scheme, a host, a path and a query string.

2) Web Server

1. What is a web server?

Is a program that uses HTTP (Hypertext Transfer Protocol).

2. What does a web server?

It serves the files from Web pages to users which are forwarded by their computers' HTTP clients.

3. How can content be stored on a web server?

On the Hypertext Transfer Protocol (http).

4. How is content in webpages described?

With a HTML document.

3) Web Client

1. What is a web client?

The Web client is a client-side component within the Java 2 Platform Enterprise Edition (J2EE), a distributed multi-tiered application model used for building and developing enterprise applications.

2. What does a web client?

It typically refers to the Web browser in the user's machine. It may also refer to plug-ins and helper applications that enhance the browser to support special services from the site

3. Web Client and Web Browser is the same

The web client can be said as an application or web browser which is installed in a computer and used to interact with Web servers upon user's request. It is basically a consumer application which collects processed data from servers.
